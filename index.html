Welcome to Flantic

Flantic is a multi-purpose bot.. Flantic provides a variety of feature set compared to other Premium Discord Bots.

Highlighted Features of Flantic

Say Hi to new members

Say hi to the new members on your Discord server with your custom welcome messages. Take advantage of the welcome message to inform newcomers about your server rules, topic, or ongoing events

Unique song request channel

When setting up flantic to be used via a custom player you'll be able to control the bot from a unique song request channel!

Setup Reaction Roles

Reaction roles are roles you can get by reacting to a message. These can be color roles, notification roles, and much more! You can set these up easily using flantic bot.

Host Giveaways in your server

The Giveaways module allows you to host timed giveaways which users can enter for a chance to win a prize. Users can enter these giveaways in your Discord server.

High-Quality Auto-Moderation System

Customizable moderation bot offering auto mod, custom commands, logging, word filters, and much more!
